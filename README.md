# Week 2 - Flask REST API Project

## Overview
This project is a RESTful API built using Flask to manage student data with CRUD operations.

## Features
- Add student (POST)
- Get all students (GET)
- Update student (PUT)
- Delete student (DELETE)

## Tech Stack
- Python
- Flask

## API Endpoints
- POST /students
- GET /students
- PUT /students/<id>
- DELETE /students/<id>

## How to Run
pip install flask  
python app.py  

## Testing
APIs tested using Postman (Desktop / Lightweight Client)

## Project Structure
- app.py
- models/
- routes/
- requirements.txt
