# Week 2 - Flask REST API Project

## Overview
Built a REST API using Flask to manage student data.

## Features
- Add student
- View students
- Update student
- Delete student

## API Endpoints
- POST /students
- GET /students
- PUT /students/<id>
- DELETE /students/<id>

## How to Run
pip install flask
python app.py

## Tools
- Postman for testing
